MODULE 2: JAVASCRIPT



Introduction au module:


JavaScript est bien plus qu'un simple langage de programmation; c'est le dynamisme et l'âme interactifs du web moderne. Conçu initialement pour fonctionner côté client dans les navigateurs web, JavaScript a transcendé cette limitation originelle. Sa beauté réside dans sa simplicité apparente, couplée à une profondeur et une flexibilité qui permettent aux développeurs d'exprimer leur créativité sans contraintes. Qu'il s'agisse d'ajouter un simple effet de survol sur un bouton, de gérer des formulaires complexes ou de créer des applications web à page unique, JavaScript est à la base de tout cela.


Ce module est consacré à l'un des langages de programmation les plus influents et omniprésents du web : JavaScript. De son origine modeste dans les années 1990 comme simple outil pour dynamiser les pages web, JavaScript a évolué pour devenir une force motrice, façonnant l'expérience utilisateur et la logique métier des applications web modernes. Ce module a été soigneusement structuré pour aborder les deux dimensions majeures de JavaScript dans le développement web:


La partie "FrontEnd": JavaScript


Dans la première section de notre module sur le "FrontEnd" en JavaScript, nous explorerons le cœur du développement web côté client.
Le "FrontEnd" est l'endroit où la magie visuelle et l'interaction utilisateur prennent vie, et au centre de tout cela se trouve JavaScript.

Cette section se subdivise en trois volets essentiels, chacun apportant sa propre dimension au développement web :
  • Le langage JavaScript

  • La librairie ReactJS

  • Le framework Angular


La partie "Back Office": Java Entreprise Edition (JEE)


la deuxième partie de notre module dédié au développement Java, orientée vers le "Back Office" et s'appuyant sur la puissante Java Entreprise Edition (JEE). Dans ce volet, nous plongerons dans le monde sophistiqué de la conception, de la mise en œuvre et du déploiement d'applications Java d'envergure.

Cette section comprend un ensemble de thèmes clés, chacun étant un maillon essentiel de l'écosystème JEE :
  • Introduction à JEE, JSP, EJB et Web services

  • Java Server Faces (JSF)

  • Architecture distribuée et JEE

  • Tests, intégration et déploiement

  • Sécurité et gestion des erreurs

  • Le Framework Struts

  • Le Framework Spring

  • Création de services avec TypeScript

  • Projet JEE : Conception, réalisation et déploiement d'une application JEE d'envergure
